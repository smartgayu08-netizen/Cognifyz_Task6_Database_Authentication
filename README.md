# Cognifyz Task 6 - Database Integration and User Authentication

## Objective

Integrate a database and implement user authentication for secure data handling.

## Features

* User Registration
* User Login
* Password Hashing using bcrypt.js
* Session Management
* Protected Dashboard Route
* Logout Functionality
* MongoDB Database Integration

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* bcrypt.js
* express-session
* CSS

## How to Run

1. Install dependencies

npm install

2. Start MongoDB

3. Run the application

node app.js

4. Open Browser

http://localhost:3000
